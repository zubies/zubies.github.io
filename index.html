Projects

Vehicle Routing Simulation

About of the project: Simulator for multiple modes of transportation dependent on an array of variables. Want to send a bike through the mountians with 300lb backpack? You sure can.

Functional requirements/what the project does: The project takes multiple variables such as road condition, mode of transport, payload, distance, and route and creates a simulator for the best path that they could take

Project design: To start we have the abstract vehicle which contains all the most basic tasks of a vehicle. Then each unique type of vehicle ex: bike, van, semi, have their own unique characteristics that make them preferable for certain tasks. Next is the routes, there are three types of routes, city, rural, and highway. Because of the way that different vehicles interact with different routes some will be faster, slower, or cheaper than another for a given distance and route type. The last thing is the simulator which runs the calculations on which assignments should be tasked to which available transportation modes dependent on the speed and cost.
